# album-copa-2026

Projeto 100% desenvolvido com o Claude AI versão gratuita. 

Os dados do preenchimento são salvos no localstorage do browser, portanto se limpar os dados e navegação ou usar aba anônima seu dados serão perdidos. 

#Principais features
1. Preenchimento de figurinhas conquistadas
2. Controle de repetidas(basta clicar mais de uma vez em uma figurinha)
3. Busca por seleção
4. Busca rápida por figurinha
5. Modo troca com envio de mensagem pelo Whatsapp das figurinhas repetidas
6. Progresso por seleção e grupo
7. Cards informativos no topo do app
8. Reports ao clicar nos cards Tenho, Faltam e Repetidas
9. Relatórios exportáveis
10. Mapa Mundi
